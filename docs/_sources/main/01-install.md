# 배포

Virton(이하 버트온)은 더품에서 출시한 Proxmox 기반 가상화 솔루션입니다.<br>
여러 기업을 대상으로 가상화 시연회, PoC를 진행하여 레퍼런스를 확보하는 과정에서 한국 사용자 의견을 반영하여 UI, 관리 기능 등 개선 필요성을 느낀 것이 솔루션 출시 계기라고 볼 수 있습니다.<br><br>
버트온은 클러스터 전체 혹은 개별 노드의 CPU, 메모리, 디스크, 네트워크 등 현재 시점의 리소스 사용 현황을 통합 조회할 수 있습니다. 각 노드 간의 연결 상태와 업타임, 버전, 서비스 상태 등을 실시간으로 모니터링하고 알림 설정 및 조회가 가능합니다. <br>
현재 <point>VirtOn 1.1</point>의 주요 기능은 다음과 같습니다.
* <point>VM 관리</point>
    * 가상머신 생성, 수정, 삭제 및 콘솔 접속 기능 제공
    * 템플릿과 cloud-init을 활용하여 표준화된 환경을 빠르게 배포하고 초기 설정을 자동화 할 수 있음
* <point>로컬/공유 스토리지 조회</point>
    * 각 노드의 로컬/공유 스토리지(Ceph, iSCSI, NFS 등)의 총 용량과 사용률 조회 가능
    * 스토리지 유형, 상태, 마운트 경로, 데이터스토어별 가상 디스크 분포를 통합 화면에서 직관적으로 확인 가능
* <point>네트워크 구성 조회</point>
    * 다양한 네트워크 구성(bridge, VLAN, OVS, SDN Zone 등)을 시각적으로 파악할 수 있음
    * 노드 간의 네트워크 연결 구조와 가상 네트워크 인터페이스, 대역폭, MTU 등의 설정값도 함께 표시됨
* <point>백업 조회</point>
