1. 개요

1. 개요#

버트온(VirtOn)은 Proxmox 기반 가상화 솔루션입니다.
버트온은 클러스터와 개별 노드의 CPU, 메모리, 네트워크 등 실시간 리소스 사용률에 대한 모니터링 및 관리 기능을 고도화하고 각 노드 간의 연결 상태와 업타임, 버전, 서비스 상태 등을 통합적으로 조회•제어할 수 있도록 합니다.

현재 VirtOn 1.1의 주요 기능은 다음과 같습니다.

  • VM 관리

    • 가상머신 생성, 수정, 삭제 및 콘솔 접속 기능 제공

    • 템플릿과 cloud-init을 활용하여 표준화된 환경을 빠르게 배포하고 초기 설정을 자동화

  • 로컬/공유 스토리지 조회

    • 각 노드의 로컬/공유 스토리지(Ceph, iSCSI, NFS 등)의 총 용량과 사용률 조회 가능

    • 스토리지 유형, 상태, 마운트 경로, 데이터스토어별 가상 디스크 분포를 통합 화면에서 직관적으로 확인 가능

  • 네트워크 구성 조회

    • 다양한 네트워크 구성(bridge, VLAN, OVS, SDN Zone 등)을 시각적으로 파악할 수 있음

    • 노드 간의 네트워크 연결 구조와 가상 네트워크 인터페이스, 대역폭, MTU 등의 설정값도 함께 표시됨

  • 백업 스케줄 조회 및 백업 카탈로그 관리

    • vzdump 기반 혹은 Proxmox Backup Server를 통해 백업된 VM/CT의 목록, 백업 시점, 백업된 용량, 수행 결과(성공/실패) 로그를 카탈로그 형태로 조회

    • 각 백업 데이터에 대한 세부 정보 및 복구 옵션 지원

    • 이를 통해 백업 주기, 실행 상태, 스토리지 사용량을 효율적으로 파악하고 신속한 복구 작업을 수행할 수 있음

  • 로그인 설정

    • 별도의 인증 절차 없이 Proxmox 사용자 계정을 통해 버트온에 직접 접근 가능하여, 통합 계정 기반으로 클러스터 및 내부 리소스 관리 가능

  • 이벤트/알람 기능

    • 클러스터, 노드, VM, 스토리지, 백업 등 주요 구성 요소에서 발생하는 모든 이벤트를 실시간으로 수집•출력

    • 장애나 리소스 한계 상황이 감지될 경우 관리자에게 즉시 알림을 전송하며, 모든 이벤트는 로그로 저장되어 이후 추적 및 분석 가능
