3. 연결 및 구성#

정상적으로 버트온 웹페이지에 접속되었다면 그 다음에 해야하는 설정은 사용 중인 Proxmox 노드를 연결하는 것입니다.
아래 단계를 수행하여 운영 중인 Proxmox 클러스터 및 노드를 연결하세요.

  1. 왼쪽의 설정 탭으로 이동합니다.

  2. 연결할 Proxmox 호스트 정보를 입력합니다.

    • 호스트 : Proxmox에 접근하기 위한 IP주소 혹은 도메인을 입력

    • 포트 : Proxmox API는 HTTPS 프로토콜을 사용하고, 기본값은 8006

    • 사용자 이름 : 로그인할 계정을 입력(예: root@pam)

    • 비밀번호 : 위 로그인 계정의 비밀번호를 입력

  3. 입력한 정보를 확인한 후, 아래 연결테스트 버튼을 클릭하여 통신이 가능한지 확인합니다.

![](../_static/images/alert.svg) 연결 실패된 경우, 입력한 정보를 다시 확인한 다음 연결테스트를 진행합니다.

  1. 통신이 가능한 경우, 설정 저장을 눌러 호스트 정보를 저장합니다.
